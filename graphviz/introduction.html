はじめに

GraphViz とは

GraphViz とは決められたフォーマットのテキストファイルをダイアグラムに変換するソフトウェアです。どのようなダイアグラムを描くことができるのかは公式サイトのギャラリーを参照してください。下のようなダイアグラムがたくさん例示されています。

GraphViz は以下の OS で動作します。

  • MS-Windows
  • MacOS
  • Linux ディストリビューション
  • Solaris
  • BSD 系 OS

このサイトでは MS-Windows と Linux 環境で GraphViz を使うことを前提として書いていきます。

MacOS を取り上げないのは単に筆者が Mac を持っていないためです。ですがこのサイトに書かれた内容は GraphViz のインストールを除いて Mac でもそのまま使うことができるはずです。
