GraphViz のインストール

早速ですが GraphViz のインストール方法を OS ごとに説明していきます。それぞれの OS ごとに以下の手順に従ってインストールしてください。

MS-Windows
  1. GraphViz 公式サイトのダウンロードセクションから最新のインストーラをダウンロードしてください。
  2. ダウンロードしたインストーラを実行してインストーラの指示に従ってください。
Linux ディストリビューション

各 Linux ディストリビューションのリポジトリの GraphViz が最新バージョンであればそれをディストリビューションのパッケージマネージャでインストールしてください。

最新でなければソースが GraphViz 公式 GitLab リポジトリで公開されていますので、それをビルドしてください。ビルド方法は tar.xz もしくは tar.gz ファイル内の INSTALL ファイルに書かれていますので、それに従ってください。

なお Web で最新の GraphViz のバージョンを検索して調べるとやたらと古いバージョンが最新だという情報がヒットします。2025 年 4 月 26 日現在の GraphViz の最新バージョンは 12.2.1 です。chatGPT などのチャット AI も平気でやたらと古いバージョンを最新であると言い張るので注意してください。

その他の OS

GraphViz 公式サイトのダウンロードセクションにインストール方法が書かれていますのでそれに従ってください。
